## Встановлення та запуск
1. Клонуйте репозиторій:
   ```sh
   git clone <repo_url>
   ```
2. Перейдіть у директорію проєкту:
   ```sh
   cd project_folder
   ```
3. Встановіть необхідні залежності:
   ```sh
   npm install  # або yarn install
   ```
4. Запустіть проєкт:
   ```sh
   npm start  # або yarn start
   ```
